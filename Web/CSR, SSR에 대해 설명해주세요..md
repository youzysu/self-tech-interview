#### 페이지 렌더링 방식

|  |  CSR | SSR |
| :-: | :-: | :-: |
| 페이지 **렌더링 위치** | 클라이언트 <br /> (브라우저) | 서버 |
| 초기 **로딩** 속도 | 👍 | 👎 |
| 초기 **렌더링** 속도 <br /> (TTV) | 👎 | 👍 |
| 초기 **인터랙션** 속도 <br /> (TTI) | 👍 | 👎 |
| 검색 엔진 최적화 **SEO** | 👎 | 👍 |
| **페이지 이동** <br /> (깜빡거림) | ❌ <br /> SPA | ⭕️ <br /> MPA |

## CSR
- Client-Side Rendering
- 클라이언트(브라우저)에서 동적으로 HTML Element를 생성하여 페이지를 렌더링하는 방식

## SSR
- Server-Side Rendering
- 서버에서 페이지 렌더링을 진행하고 클라이언트에 전달하는 방식
